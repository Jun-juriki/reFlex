# 🐧 reFlex - Персонализированная AI-платформа для реабилитации

**reFlex** — это инновационная веб-платформа для персонализированной реабилитации пациентов с нарушениями опорно-двигательного аппарата, после инсульта и травм. Платформа использует AI (GigaChat) для генерации SMART-целей, программ тренировок и мотивационных советов, а также включает игровые механики для повышения вовлечённости.

## 🎯 О проекте

**reFlex** (Rehabilitation + Flexability) создан для того, чтобы сделать процесс восстановления:
- **Персонализированным** — AI адаптирует программу под каждого пациента
- **Мотивирующим** — игровые механики (XP, уровни, серии, достижения)
- **Прозрачным** — наглядные графики прогресса и прогнозы
- **Доступным** — веб-платформа с мобильной адаптацией

### Для кого этот проект?
- Пациенты, проходящие реабилитацию после травм, инсультов, операций
- Врачи-реабилитологи (для мониторинга прогресса пациентов)
- Медицинские центры и клиники

## ✨ Ключевые возможности

### 🤖 AI-функционал (GigaChat)
- **Генерация SMART-целей** на основе диагноза и текущих показателей
- **Создание персонализированной программы тренировок** с 3 уровнями сложности
- **Мотивационные AI-советы** с учётом прогресса и серии


### 🎮 Геймификация
- **XP и уровни** — каждые 100 XP = новый уровень ??
- **Серии (Streak)** — дни подряд с тренировками
- **Достижения (Achievements)** — награды за прогресс
- **Квесты** — ежедневные и долгосрочные задания
- **Мотивационный пингвин** — случайные ободряющие фразы

### 📊 Отслеживание прогресса
- **Прогноз достижения цели**
- **Интерактивный график** «План vs Факт» (Chart.js)
- **Визуальная карта восстановления**
- **История активности и начисленных XP**

### 🏋️ Программа тренировок
- 3 упражнения: разминка, основное, закрепление
- 3 уровня прогрессии нагрузок
- Блокировка упражнений на время восстановления
- Подробные инструкции по технике выполнения

### 🔐 Аутентификация и профиль
- Регистрация/вход по email + пароль
- Личный кабинет с полной статистикой
- Медицинская анкета для первичного опроса

## 🚀 Установка и запуск

### Предварительные требования
- Python 3.10 или выше
- Git


### Шаг 1. Клонирование репозитория

```bash
git clone
cd reFlex
```

---

## Шаг 2. Создание виртуального окружения
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Mac/Linux
python3 -m venv venv
source venv/bin/activate
```
---

## Шаг 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

---

## Шаг 4. Настройка переменных окружения
Создайте файл .env в корне проекта:

```env
# GigaChat API
GIGACHAT_CREDENTIALS=ваш_ключ_авторизации
GIGACHAT_SCOPE=GIGACHAT_API_PERS
GIGACHAT_VERIFY_SSL_CERTS=False
```

---

## Шаг 5. Миграции базы данных
bash
python manage.py makemigrations
python manage.py migrate

---

## Шаг 6. Создание суперпользователя (для админ-панели)
```bash
python manage.py createsuperuser
```
---

## Шаг 7. Запуск сервера разработки
```bash
python manage.py runserver
```
---

## Шаг 8. Открыть в браузере
```text
http://127.0.0.1:8000/
```

## 🔌 API эндпоинты
| Эндпоинт | Метод | Описание |
|----------|-------|----------|
| `/` | GET | Главная страница |
| `/register/` | GET, POST | Регистрация нового пользователя |
| `/login/` | GET, POST | Аутентификация и вход в систему |
| `/profile/` | GET | Личный кабинет с прогрессом и статистикой |
| `/anketa/` | GET, POST | Медицинская анкета для сбора данных пациента |
| `/exercise/<int:idx>/` | GET | Детальная страница упражнения |
| `/exercise/<int:idx>/complete/` | POST | Отметка выполнения упражнения (успех/неудача) |
| `/api/generate-goal/` | POST | Генерация SMART-цели через AI (AJAX) |
| `/api/get-advice/` | GET | Получение персонализированного AI-совета |
| `/api/log-exercise/` | POST | Логирование выполнения упражнения |
| `/api/exercise-progress/` | POST | Отправка прогресса по упражнению |
| `/api/generate-instruction/` | POST | Генерация инструкции к упражнению через AI |

## 🗄 База данных
### Основные модели

| Модель | Поля | Описание |
|--------|------|----------|
| **User** (core/models.py) | `email`, `full_name`, `age`, `gender`, `password` | Пользователь системы (аутентификация) |
| **PatientProfile** (core/models.py) | `user` (OneToOne), `diagnosis`, `limitation_type`, `baseline_value`, `target_value`, `target_days`, `goal_text`, `training_program` (JSON), `strikes`, `max_strikes`, `current_progress`, `total_xp`, `level` | Медицинский профиль и прогресс пациента |
| **ActivityLog** (gamification/models.py) | `user`, `exercise_name`, `status` (success/failed), `completed_at`, `next_available_at` | Логи выполнения упражнений |
| **UserStats** (gamification/models.py) | `user` (OneToOne), `total_xp`, `level`, `current_streak`, `max_streak`, `last_activity_date`, `current_progress`, `current_value` | Статистика геймификации пользователя |
| **Achievement** (gamification/models.py) | `name`, `slug`, `icon`, `description` | Список доступных достижений |
| **UserAchievement** (gamification/models.py) | `user`, `achievement`, `earned_at` | Привязка достижений к пользователям |

## 🤖 AI интеграция (GigaChat)
Настройка
1. Зарегистрируйтесь в кабинете разработчика Сбера
2. Создайте приложение и получите credentials
3. Добавьте GIGACHAT_CREDENTIALS в .env

## Методы AI-сервиса
```python
# core/ai_service.py

ai_service.generate_smart_goal(profile)
# → "Увеличить дистанцию ходьбы с 50 до 150 метров за 28 дней"

ai_service.generate_exercise_program(profile)
# → [{"name": "Разминка", "description": "...", "progression": {...}}]

ai_service.generate_advice(progress_percent, days_diff, streak, level)
# → "🔥 Отличная работа! Продолжай в том же духе!"
Fallback-режим
Если GigaChat API недоступен, автоматически используются встроенные шаблоны.
```
